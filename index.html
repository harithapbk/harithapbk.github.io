Projects

112 ERSS Analytics Dashboard(CDAC)

Developed a nationwide 112 call service Analytics Dashboard using Python, Plotly, folium, HTML, CSS, Django, and pandas. Built crime pattern predictive models with TensorFlow, Scikit-learn, Python, and Django Rest-API. Collaborated on defining needs, integrating data, and providing real-time visualizations for emergency service administrators.

Police Patrol Route prediction(CDAC)

Implemented patrol route optimization based on predicted crime hotspots using Evolutionary Algorithms, geopandas, and the shapely library. Utilized GPU for accelerated processing of route optimization and waypoint assignment based on hotspots.

  • Python
  • geopandas
  • scikit-learn
  • shapely

End-to-end big data analytics pipeline (CDAC)

Established a comprehensive big data pipeline for ingesting, processing, and analyzing data from the 112 service. Leveraged PostgreSQL, Apache NiFi, Apache Kafka, Apache Spark, Hadoop HDFS, and Hive for data processing and analytics.

  • Pyspark
  • Hadoop
  • Apache NiFi
  • Hive
  • Kafka

Natural Language queries to SQL(CDAC)

Implemented a natural language query interface for generating SQL-based 112 Analytics reports. Leveraged Ollama and Llama2 LLM models to convert user's natural language queries into SQL queries, allowing users to interactively create reports.

  • Python
  • Ollama
  • Llama2
